# 🍎 AI-Powered Food Recognition and Recommendation System

**Final Year Project** - Food Expiry Tracking and Recipe Recommendation System

## 📋 Project Overview

An intelligent system to reduce household food waste by:
- Tracking food expiry dates (manual, barcode, image, OCR)
- Sending timely alerts (browser + email)
- Recommending recipes using expiring ingredients
- Providing personalized diet recommendations
- AI-powered chatbot for cooking queries
- Food waste tracking and analytics

## 🔧 Tech Stack

- **Frontend**: Streamlit (Python)
- **Backend**: FastAPI (Python)
- **Database**: SQLite
- **ML**: YOLOv8 (food recognition), Tesseract (OCR)
- **APIs**: Edamam, Google Gemini, Open Food Facts, Mailgun
